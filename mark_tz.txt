Разработка интерактивного приложения для аэропорта в онлайн режиме.


Главная страница
 1.1 Вкладка пользователя.
     При нажатии на данную вкладку, открывается окно с актуальной информацией о купленных билетах, с контактной информацией(-ях): ФИО, номер рейса и места. 

 1.2 Вкладка вылет.
     При нажатии на данную вкладку, откроется отдельная страница с таблом рейсов, вылетающих из аэропорта.
     Можно будет просмотреть расписание как и актуальных рейсов, так и улетевших рейсов. Данное табло будет генерироваться автоматически с периодичностью +-1.5-3 мин. (номер рейса, город назначения, статус).
     Также в поле поиска можно указать ключевое слово для поиска рейса по нему, например (г. назначения, н. рейса и т.д) 
 
 1.3 Вкладка прилет.
     При нажатии на вкладку прилет, откроется отдельная страница с таблом рейсов, прилетающих в аэропорт.
     Можно будет просмотреть расписание как и актуальных рейсов, так и прилетевших рейсов. Данное табло будет генерироваться автоматически с периодичностью +-1.5-3 мин.(номер рейса, город назначения, статус).
     Также в поле поиска можно указать ключевое слово для поиска рейса по нему, например (г. назначения, н. рейса и т.д) 
     
 1.4 Покупка/Возврат билета
     Данная вкладка вызывает страницу, на которой пользователю выводится табло рейсов, вылетающих из аэропорта, а также поля,на которых предлагается ввести номер рейса, ФИО, номер места.
     Кол-во занятых мест также будет генерироваться автоматически. Для покупки билетов на несколько человек, нужно проделывать операцию заново указывая новые данные. Цена 1 авиабилета будет рассчитываться исходя из данных рейса.
     При изменении статуса рейса на вылетел(улетел), а также при условии что пользователь не вернул билет(-ы), приложение закроется.
     Возврат билета не будет осуществляться при условии, что времени до вылета рейса осталось 1 мин. или менее.
     
 1.5 Поле времени
     На данном поле будет отображаться актуальное время.
 
 1.6 Поле погоды
     На данном поле будет отображаться состояние погоды в словесной форме.
     
 1.5 Услуги
     Данная вкладка вызывает страницу, на которой находится общая информация об услугах аэропорта с картинками, например : Парковка, Гостиница, Duty Free, Аптека, Магазины, кафе и т.д.
     При нажатии на любую из услуг, открывается соответствующая страница с описанием данной услуги, информацией для связи с персоналом.
